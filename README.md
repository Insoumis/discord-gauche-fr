# React + Vite

Ce projet utilie les templates React et Vite.js

## TailwindCSS v4.1

Pour la mise en page de ce projet, le template TailwindCSS est utilisé, il offre un avantage conséquent en terme d'animation du CSS, et de praticité. 
Afin d'insérer du CSS ou de le modifier, vous devrez aller dans la partie __ClassName__ de chaque borne. 

Fiez vous à la [Documentation de Tailwind](https://tailwindcss.com/docs/)
